📈 Dave’s Medium Blog Repository Library
Welcome! This is where I keep the "engine room" for my Medium Blog.

This repository is a collection of the Jupyter Notebooks and Python scripts I’ve built while diving deep into the world of quantitative finance. I started this project as a way to document my learning process and share the math behind the models that intrigue me most. Whether it’s deriving classic equations or testing out modern volatility forecasting, you’ll find the code and logic right here.

📂 Project Focus
Each folder in this repo corresponds to a deep-dive article I've written. Here is what I’m currently working on:

Volatility Modeling: Implementing the Yang-Zhang Estimator and the HAR-RV-X Model for high-frequency forecasting.

Stochastic Calculus: A step-by-step Derivation of the Black-Scholes Model and practical applications of Market Implied Volatility.

State-Space & Regime Models: Using Kalman Filters, Hidden Markov Models (HMM), and Statistical Jump Models for regime classification.

Strategy & Theory: Analyzing Trend Following for CTAs, the Random Walk Hypothesis, and Markov Chains for corporate bond pricing.

The Modern Market: Quantitative methods for Prediction Markets and a data-driven look at the social media-driven popularity of Day-Trading.

Math & Tools: Maximum Likelihood Estimation (MLE) for parameters, Neural Networks (NLP) for keyword extraction, and creating Interactive Graphics for model visualization.
