# 📈 Dave’s Medium Blog Repository Library

Welcome! This is the central hub for the "engine room" of my [Medium Blog](https://medium.com/@2115hsun). 

This repository is a collection of the Jupyter Notebooks and Python scripts I’ve built while diving deep into the world of quantitative finance. I started this project to document my learning process and share the math behind the models that intrigue me most. Whether it’s deriving classic equations or testing out modern volatility forecasting, you’ll find the code and logic right here.

---

## 📂 Project Focus
Each folder in this repo corresponds to a deep-dive research piece or article. Here is what I’m currently exploring:

### Volatility Modeling & Forecasting
* **Yang-Zhang Volatility Estimator**
* **HAR-RV-X Model:** Using Heterogeneous Autoregressive models with exogenous variables for realized volatility.
* **Statistical Jump Models:** Exploring regime classification and discrete shifts in market dynamics.

### Stochastic Calculus & Derivatives
* **Deriving the Black-Scholes Model:** A first-principles approach using Itô's Lemma and GBM.
* **Market Implied Volatility:** Analyzing the surface and Greeks within the Black-Scholes framework.
* **Maximum Likelihood Estimation (MLE):** Calibrating parameters for random variables in SDEs.

### State-Space & Markov Processes
* **Kalman Filter:** Optimal state estimation for tracking noisy financial signals.
* **Hidden Markov Models (HMM):** Identifying latent market states and potential model improvements.
* **Markov Chains:** Application to corporate bond transition matrices and credit risk.

### Strategy, Theory & Modern Markets
* **Trend Following for CTAs:** Evaluating strategies against the **Random Walk Hypothesis**.
* **Prediction Markets (Polymarket):** Quantitative methods for identifying arbitrage in decentralized markets.
* **NLP for Finance:** Using Neural Networks for keyword extraction in financial text.
